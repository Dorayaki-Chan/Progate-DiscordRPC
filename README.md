# ProgateDiscordRPC

chromeドライバーを使って閲覧中のサイトのURLを取得し、そのURLがProgateの場合、どの言語を勉強しているのか推測し、その情報をDiscordでRichPresenceします。弄ってたら何か成功しちゃった系コードなので、どこまで必要でどこが不要のかよくわかってないです。
